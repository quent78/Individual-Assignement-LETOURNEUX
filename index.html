Visualization of the world population versus the temperature changes from 1960 to 2015

Interpretation

Making a linear regression between the world population and the temperature changes allows us to see the strong linear relationship between both (correlation coefficient of 0.94 and very small p-value). Moreover, this jointploint gives us the histograms and density plots of the two series, we notice a similar distribution between the world population and temperature changes.

Discussion

What I like with this viz is the fact we have different complementary informations in the same graph. It allows to clearly find a linear relationship between the two series. The main challenges were to clean the data (date format, extract the year, merge, filter) and to understand how to personalize each elements of the jointplot (colors, size, backgrounds, positions, title, source...). When you understand you have to conbine seaborn and matplotlib to manage the different elements of your viz, it becomes very interesting.

Code : import pandas as pd import seaborn as sns import numpy as np import matplotlib.pyplot as plt from datetime import datetime as dt #Import CSV population file pop = pd.read_csv("world_population.csv") #Convert the Year column in date format and keep the year pop['Year'] = pd.to_datetime(pop['Year'], format ='%Y') pop['Year'] = pop['Year'].dt.year pop.head #Import CSV temperature change file temp = pd.read_csv("global-temperature-1.csv") #Convert the date column in date format and keep the year temp['date'] = pd.to_datetime(temp['date']) temp['Year'] = temp['date'].dt.year #Make a filter in order to keep only the annual mean of each year temp = temp[temp['value_type'].str.contains('annual_mean') == True] #Only keep the fields we need temp = temp[["Year","value"]] #Rename the column value temp.columns = ['Year','Temperature change'] temp.head #Merge the two dataframe using inner joint on the Year master = pd.merge(pop, temp, how='inner',on='Year') master.head #Viz construction sns.set_style('darkgrid') #Dark background g = sns.jointplot(x=master['Population'], y=master['Temperature change'], data=master, kind="reg", color="c", ratio=3, joint_kws={'color':'royalblue'}) g.ax_marg_x.set_facecolor('white') #White background for the density/histo plots g.ax_marg_y.set_facecolor('white') plt.suptitle('Population vs temperature changes (1960-2015)', size=18, y=1.08, x=0.42) #Title and its position plt.xlabel("World population", size=14) #x and y label and their size plt.ylabel("Temperature changes", size=14) plt.annotate('Source: World Bank & Goddard Institute for Space Studies', (0,0), (10, -60), xycoords='axes fraction', textcoords='offset points', va='top') #Add the source plt.savefig('Population vs temperature changes.png', bbox_inches='tight') #output Visualization on the link
